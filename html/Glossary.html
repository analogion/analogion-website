Βόμβος - Vomvos

A continuous sound with some specified pitch. Usually taken as the note Νη.

Βάση παραγωγής - Production basis

A specified note that serves as a reference for mode production. There are two generally accepted ones:
Νη (the so called first mode production basis) and Δι (the so called second mode production basis).

Παραγωγή - Mode Production

By moving up or down from a basis of mode pruduction we succesively find the bases of other modes.
For instance moving up from Νη we find Πα (first mode), Βου (second mode), Γα (third mode), Δι (fourth mode).
And moving down from Δι we find Γα (third plagal), Βου (second plagal), Πα (first plagal), Νη (fourth plagal).

Σύστημα - System

A set of defined intervals (e.g. a tetrachord or a pentachord or an octave)
repeating up and down unchanged. The last note of the last repetition becomes the first note of the next repetition.
See for example the pentachordon system.

Πεντάχορδο Σύστημα (Τετραφωνία) - Pentachordon System

A type of system. A set of 4 defined intervals (a pentachord) repeating up and down unchanged.

E.g. the following pentachord (set of four intervals enclosed within five notes)

Πα -Βου-Γα-Δι- Κε
can be repeated as

... Πα -Βου-Γα-Δι- Κε(Πα)-Βου-Γα-Δι- Κε(Πα)-Βου-Γα-Δι- Κε(Πα)-...

to form the Pentachord system.
Οκτάχορδο Σύστημα (Διαπασών) - Octachordon System

A type of system. A set of 8 notes (7 defined intervals, an octave) repeating up and down unchanged.

Τετράχορδο Σύστημα (Τριφωνία) - Tetrachordon System
A type of system. A set of 3 defined intervals (a tetrachord) repeating up and down unchanged.

Μέσος Ήχος - Mesos (middle) mode

The mode that its basis is two notes below the basis of the related authentic mode

Παράμεσος Ήχος - Paramesos mode

The mode that its basis is three notes below the basis of the related authentic mode

Πλάγιος Ήχος - Plagal mode

The mode that -as a rule- its basis is four notes below the basis of the related authentic mode.

Αντίφωνος Ήχος - Antiphonos mode

The mode that its basis is seven notes (one octave) below the basis of the related authentic mode

Δίφωνος Ήχος - Diphonos mode

The mode that its basis is two notes above the basis of the related plagal mode

Τρίφωνος Ήχος - Triphonos mode

The mode that its basis is three notes above the basis of the related plagal mode

Τετράφωνος Ήχος - Tetraphonos mode

The mode that its basis is four notes above the basis of the related plagal mode

Πεντάφωνος Ήχος - Pentaphonos mode

The mode that has the same basis as the related plagal but its range is mainly within five notes above the basis of the related plagal mode.
And the melody likes to exhibit the interval between the basis and the 6th note of the scale.

Επτάφωνος Ήχος - Heptaphonos mode

The mode that its basis or its range is seven notes above the basis of the related authentic mode.

Μικτός Ήχος - Mixed mode

A mode that is made up from tetrachords/pentachords of other modes, usually of different genres (e.g. diatonic-chromatic, diatonic-enharmonic etc.).
An example is Fourth Plagal chromatic made up from a diatonic pentachord and a soft chromatic tetrachord.

Μακάμ - Maqam (or Makam)
The equivalent of a mode in Eastern secular music. Makams have Persian/Arabic/Turkish names.

Μόριο (πλ. Μόρια) - Morion/Cent (pl. Moria/Cents)

The elementary interval unit.

Μείζων (Μείζονας) Τόνος - Major diatonic tone interval

An interval that consists of 12 moria.

Ελάσσων (Ελάσσονας) Τόνος - Minor diatonic tone interval

An interval that consists of 10 moria.

Ελάχιστος Τόνος - Smallest diatonic tone interval

An interval that consists of 8 moria.

Ημιτόνιο - Half a tone

An interval that consists of 6 moria.

Τριτημόριο του μείζονος τόνου - One third of a tone

An interval that consists of 4 moria.

Τεταρτημόριο του μείζονος τόνου - One fourth of a tone

An interval that consists of 3 moria.
